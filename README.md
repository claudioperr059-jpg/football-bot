# FOOTBALL BOT 🤖⚽

Bot per pronostici calcio con **25 regole** e capacità di **auto-apprendimento**.

## Regole incluse

1. No quote sotto 1.50
2. Forma reale > qualità rosa
3. Fattore casa obbligatorio
4. Motivazioni reali
5. Dopo sconfitta pesante
6. Bestie nere H2H
7. Grande occasione
8. Regressione xG
9. Regressione difensiva
10. Campionati esotici
11. No aggiunte senza ok
12. Trappole da Champions
13. Scontri diretti obiettivi
14. Squadre in crisi
15. Verifica BTTS %
16. Forma batte storia
17. Vantaggio killer
18. Crisi + goleada
19. Multipla max 3
20. Salvezza in trasferta vs big
21. Nuovo allenatore
22. 5+ infortuni
23. Difesa solida ≠ Under
24. Striscia positiva inattesa
25. X + Under 1.5 contraddittoria
